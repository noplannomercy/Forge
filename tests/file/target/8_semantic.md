## 한국교육학술정보원(KERIS) 기반 지능형 컨택센터 구축 및 운영

### 개요

기존 IPCC(Internet Protocol Contact Center)를 AI 기반 컨택센터(AICC)로 발전시켜 혁신적인 고객 경험을 제공하는 것을 목표로 한다. 핵심은 사용자 요청 처리 및 응답 생성 자동화이다.

### 핵심 아키텍처 및 구성 요소

**1. Data Preprocessing Layer:**

*   수집된 데이터 정제 및 구조화 담당
*   Text Data Cleaning/Normalization/Categorization
*   Meta Data 추출 및 생성
*   Chunking Data 처리

**2. User Action Layer:**

*   사용자 요청 처리 담당
*   Speech Processing Engine: 음성-텍스트 변환(STT/TTS)
*   LLM Processing Engine: 지능형 응답 생성 및 처리
*   Quality Control Engine: 응답 품질 관리 및 피드백
*   ChatBot: 상담 Q&A 지원, 질의 카테고리 및 FAQ 제시

**3. Knowledge Base:**

*   벡터 데이터 기반 지식 저장소 구축
*   LLM Models: 활용형 LLM Agent 구성
*   VectorDB: 벡터 데이터베이스 및 인덱스 관리
*   QMS/KMS: Quality Mgt, Data Mgt 등으로 KMS 연동
*   Error/Logging: 데이터 오류 및 로그 관리

**4. Contact Management Systems:**

*   우선 호출 라우팅 (대기시간 관리, 호출 Agent 관리)
*   상담 이력 및 로그 데이터의 체계적 관리
*   화상 상담 내용 저장 및 처리(Voice, Text)

**데이터 흐름 (다이어그램 설명):**

1.  **Data Preprocessing Layer**: 데이터 정제, 구조화, 임베딩 벡터 생성
2.  **Knowledge Base**: 벡터 DB를 통해 LLM 모델을 활용하여 유사도 기반 검색 수행
3.  **User Action Layer**:
    *   사용자 음성 요청은 Speech Processing Engine(STT)으로 텍스트 변환
    *   텍스트 요청 및 변환된 텍스트는 쿼리 처리 후 LLM 결과를 검색
    *   검색 결과는 ChatBot을 통해 사용자에게 응답 제공 (텍스트 또는 TTS를 통한 음성)

**5. LLM Management:** LLM 성능 모니터링 및 추적

**6. Logging:** 컨테이너 로그 수집 및 관리

**7. VectorStore Management:** 인덱스를 이용한 벡터 데이터베이스 관리

**8. Cache Management:** Cache format 및 Source 관리

**9. Contact Management:** Agent 및 큐 관리

**10. IPCC/CTI Recording:** 통화 녹취 데이터 관리

### CTI 기반 지능형 컨택센터

CTI(Computer Telephony Integration) 시스템을 기반으로 다양한 채널의 고객 요청을 AI 기술과 연계하여 효율적인 상담 서비스를 제공한다.

*   **멀티 채널 통합 관리:** 음성, 채팅, 상담 요청 통합 관리
*   **상담 통계:** 실시간 응대 통계 및 피드백 제공, AI 기반 상담 팁 제공
*   **지능형 상담:** AI 기반 자동 응답 및 실시간 상담 지원
*   **품질 관리:** AI 기반 모델링 및 분석

### LLM 활용 지능형 시스템

다양한 형태의 데이터를 수집하고 처리하여 정확하고 신속한 지능형 응답을 생성하는 시스템이다.

*   **지능형 데이터 처리:**
    *   실시간 데이터 수집/정제, 고품질 데이터 확보
    *   교육 도메인 특화 지식 구축
    *   정방형 DB 활용 및 실시간 영역 및 품질 향상
*   **데이터 처리 최적화:**
    *   지식 데이터 자동 업데이트
    *   컨텍스트 기반 임베딩 고도화 및 사용자 의도 파악
    *   개시 시스템 운영, 문맥 결론에 대한 응답 속도 최적화

#### AI 엔진

다양한 AI 엔진을 통합 운영하여 고객에게 고도화된 컨택센터 경험을 제공한다.

*   **Speech Processing Engine:**
    *   고품질 한국어 음성 처리, 높은 정확도로 실시간 응답 생성
    *   자연스러운 대화 흐름 지원
*   **LLM Processing Engine:**
    *   교육 특화 언어 모델 활용
    *   문맥 기반 정확한 답변 생성 및 사용자 질문 의도 파악
*   **Quality Control Engine:**
    *   응답 정확도 검증
    *   자동 피드백 생성

### KERIS의 사용자 경험 극대화

KERIS의 교육 서비스 데이터를 활용하여 사용자 경험을 개선하고, 데이터 기반 지능형 교육 행정 체계를 구축하는 데 기여한다.

**1. 운영 효율성 극대화**

*   24/7 실시간 응답 처리
*   자동화 답변으로 운영 비용 절감
*   상담 업무 효율화

**2. 교육 정보 서비스 고도화**

*   맞춤형 학습 정보 제공
*   교육 데이터 통합 관리
*   고품질 교육정보 제공

**3. 교육 현장 지원 강화**

*   신속한 정책 전달
*   현장 의견 수렴
*   교육 과정 체계 개선

**4. 교육 현장 접근성 강화**

*   IPCC/CTI 시스템 연동
*   KERIS 데이터 연계
*   자동 응답/챗봇 엔진 최적화
*   지식/콘텐츠 관리 시스템 구축
*   RAG(Retrieval Augmented Generation) 시스템 구축

**5. 서비스 품질 향상 및 확장성 확보**

*   실시간 응답 분석 및 예측 개선
*   사용자 피드백 기반 시스템 고도화
*   보안 강화 및 개인정보보호 체계 고도화
*   통합 관리 대시보드 구축
*   비상 상황 대응 체계 수립 및 운영 매뉴얼 개발

**6. 시스템 개발**

*   실시간 품질 모니터링 시스템 개발
*   챗봇/상담원 연관 관계 분석
*   데이터 로그 분석 및 QMS 연동 체계 구축
*   상담 품질 관리 기준 및 평가 체계 마련

## 내용 입력 가이드

이 문서는 내용 입력에 대한 가이드라인을 제공합니다.

### 핵심 내용

*   강조 내용을 입력하십시오. 0000 입니다.
*   kt cs '큐톤 광고' 관련 내용

### 세부 내용 입력 방법

다음은 여러 내용 입력 방식에 대한 예시입니다.

*   내용 입력해주세요
*   내용 입력하세요
    *   내용을 입력해주세요
    *   내용을 이렇게
*   원 안에 내용을 입력하는 방식 (3개 원형 다이어그램)
*   화살표로 연결된 내용 입력 방식

### 추가 정보 입력 방식 예시

다양한 형태로 추가 정보를 입력할 수 있습니다. 예시는 다음과 같습니다.

*   여러 줄의 텍스트 입력
*   드롭다운 메뉴 형태의 입력
*   버튼 형태의 입력

### 추가 예시

*   버튼: 인사, 노무, 채용, 코로나 (각 버튼 아래 OOO 표시).
*   LOLO 텍스트 입력 칸 2개.
